# Stock Market ML Prediction Dashboard

Real-time stock analysis with machine learning predictions.

## Features
- Live stock data from Yahoo Finance
- ML-powered price predictions (5-day forecast)
- Technical indicators (RSI, MACD, SMA)
- Interactive candlestick charts
- Volume analysis
- Company fundamentals

## Tech Stack
- Streamlit
- yfinance (free data)
- scikit-learn (Random Forest)
- Plotly (interactive charts)
- TA-Lib (technical analysis)

## Deploy
Free deployment on Streamlit Cloud - no API keys needed!
